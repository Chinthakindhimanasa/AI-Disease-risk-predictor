# 🧠🏥 AI Disease Risk Prediction with Explainable AI

## 📌 Project Overview
This project predicts the risk of diseases like Diabetes using Machine Learning and explains predictions using AI Explainability techniques.

It also provides a *voice output feature* that speaks the result.

---

## 🚀 Features
- 🔍 Disease risk prediction (High / Low)
- 🧠 Explainable AI (Feature importance)
- 📊 Interactive Streamlit dashboard
- 🎤 Voice output (AI speaks result)
- 📈 Probability prediction
- 💡 Health insights visualization

---

## 🛠 Tech Stack
- Python 🐍
- Pandas & NumPy
- Scikit-learn
- Streamlit
- Matplotlib
- Pyttsx3 (Voice output)

---

## 📊 How It Works
1. User enters health data
2. ML model processes input
3. Prediction is generated
4. AI explains important factors
5. Voice output reads result
